## Timeshare and Management site for MKTMC LLC

Stack:
* Angular
* Node
* Express
* MS SQL Server

I used a layered approach such that the database access is fully decoupled from the Node routers and Express controllers.

This is a closed-source and private website, but a demo is available, as shown below.  The demo uses MongoDb instead of SQL Server.
